# Los Angeles Crime Data Analysis (2020-2023)

### Project Overview
An interactive data analysis project exploring crime patterns in Los Angeles using **Python**, **Pandas**, and **Plotly**.

### Key Insights
* **Peak Crime Hours:** Identified that 12:00 PM and 6:00 PM are high-risk periods.
* **Top Crimes:** Identity Theft and Simple Battery are the most frequent incidents.
* **Demographics:** Most victims are aged between 25 and 45.

### Tools Used
* **Python:** Data cleaning and transformation.
* **Plotly Express:** Interactive visualizations.
* **Pandas:** Data manipulation.

### How to Run
1. Clone the repo.
2. Install requirements: `pip install pandas plotly`.
3. Run the Jupyter Notebook.
